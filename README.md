# Aplikasi Perpustakaan Sederhana

Project ini dibuat untuk memenuhi tugas UAS mata kuliah Pemrograman Berbasis Web.

## Identitas Project

Nama Project: Aplikasi Perpustakaan Sederhana  
Bahasa Pemrograman: PHP Native  
Database: MySQL  
Framework CSS: Bootstrap  

## Fitur Aplikasi

- Login
- Logout
- Dashboard
- CRUD data buku
- CRUD data user
- Menu navigasi
- Tampilan menggunakan Bootstrap dan CSS
- JavaScript untuk konfirmasi hapus dan pencarian data

## Struktur Tabel

Database yang digunakan bernama:

uas_pbw_perpustakaan

Tabel yang digunakan:

## Cara Menjalankan Project

1. Copy folder project ke dalam folder htdocs XAMPP.
2. Jalankan Apache dan MySQL melalui XAMPP Control Panel.
3. Buka phpMyAdmin melalui browser.
4. Buat database dengan nama uas_pbw_perpustakaan.
5. Import file database.sql.
6. Buka aplikasi melalui browser dengan alamat:

http://localhost/uas-pbw-perpustakaan

## Penjelasan Folder

- config berisi file koneksi database.
- assets berisi file CSS dan JavaScript.
- books berisi fitur CRUD data buku.
- users berisi fitur CRUD data user.
- login.php digunakan untuk proses login.
- logout.php digunakan untuk keluar dari aplikasi.
- dashboard.php adalah halaman utama setelah login.
